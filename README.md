# GymFlow AI

Auto-generated by ETS-AI 10x.

## Idea Summary
A SaaS platform designed to help small local gyms in India manage and convert leads from WhatsApp and Instagram.

## Problem Statement
Small local gyms in India lack an efficient system to capture, track, and close leads generated through popular platforms like WhatsApp and Instagram.

## Target Persona
Indian small gym owner

## MVP Features
- WhatsApp & Instagram Direct Message Integration for lead capture
- Centralized Lead Inbox for all social media inquiries
- Lead Profile Management (contact details, source, status, notes, communication history)
- Lead Status Tracking (New, Contacted, Follow-up, Converted, Lost)
- Automated Follow-up Message Templates (WhatsApp/Instagram)
- Basic Lead Qualification (ML-powered: Hot/Warm/Cold prediction)
- Dashboard for Lead Overview and Performance Metrics
- Basic Reporting (leads captured, conversion rates by source)
- User & Staff Management with role-based access

## Tech Stack
Python (Django/Flask) for backend, React/Vue.js for frontend, PostgreSQL for database, AWS/GCP for cloud infrastructure, leveraging specific APIs for WhatsApp/Instagram integration, and Scikit-learn/TensorFlow for ML models.
